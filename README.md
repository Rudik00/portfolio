# 🎨 My Portfolio

Одностраничное портфолио-приложение, выстроенное на **FastAPI** с поддержкой русского и английского языков.

## 📋 Описание

Современное веб-приложение для демонстрации проектов разработчика. Каждый проект имеет собственную страницу с подробным описанием, технологическим стеком, архитектурой и результатами.

**Ключевые особенности:**
- 🌐 **Двуязычная поддержка** (RU/EN) с сохранением выбора в localStorage
- 🔗 **Shareable URL** — язык сохраняется в параметре `?lang=ru` или `?lang=en`
- 🎭 **Единая дизайн-система** — все проекты выглядят согласованно
- ⚡ **Быстрая загрузка** — статические файлы, нет базы данных
- 📱 **Responsive Design** — отлично смотрится на мобильных и десктопе
- 🎬 **Плавные анимации** — reveal-эффекты при загрузке

## 🛠 Стек технологий

**Backend:**
- FastAPI 0.109.0
- Uvicorn (ASGI сервер)

**Frontend:**
- HTML5 / CSS3
- Vanilla JavaScript (без фреймворков)
- Google Fonts (Yanone Kaffeesatz)
- CSS Grid & Flexbox

## 🚀 Быстрый старт

### Установка зависимостей

```bash
# Активировать виртуальное окружение
source venv/bin/activate

# Установить библиотеки
pip install -r requirements.txt
```

### Запуск приложения

```bash
# В корне проекта
uvicorn app.main:app --reload
```

Приложение будет доступно по адресу: **http://localhost:8000**

## 🌍 Использование

- **Главная страница:** http://localhost:8000/home_page
- **Проекты:** http://localhost:8000/home_page/project_1 и т.д.
- **Смена языка:** Добавить `?lang=en` в конец URL для английского языка

Пример:
```
http://localhost:8000/home_page?lang=en
http://localhost:8000/home_page/project_2?lang=ru
```

## 💡 Особенности реализации

### Двуязычная система

Каждая страница содержит:
- **textMap** — объект с переводами текста
- **setLanguage()** — функция для переключения языка
- Сохранение выбора в `localStorage` и URL-параметре

### Дизайн

- Тёмная тема с градиентными фонами
- Переиспользуемые CSS-переменные (--bg, --ink, --paper и т.д.)
- Микроанимации для улучшения UX
- Mobile-first подход

### Backend логика

- Простая раздача HTML файлов через FileResponse
- Автоматический редирект с `/` на `/home_page`
- Логирование всех запросов с временем отклика
